# Customer Event Filter

This project is an Angular application for filtering customer events. It allows users to build funnel steps with event selection and attribute filtering.

## Prerequisites

- Node.js (v18 or higher recommended)
- npm

## Getting Started

1. **Clone the repository** (if you haven't already):
   ```bash
   git clone https://github.com/08XI/bloomreach.git
   cd customer-event-filter
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

## Running the Application

To start the development server:

```bash
npm run dev
```

Navigate to `http://localhost:4200/`. The application will automatically reload if you change any of the source files.

## Running Tests

To execute the unit tests using Vitest:

```bash
npm test
```

## Building

To build the project for production:

```bash
npm run build
```

The build artifacts will be stored in the `dist/` directory.
